# CKNB Solutions — Team Portfolio

Official portfolio website for CKNB Solutions freelance dev team.

## Deploy on Render

1. Push this folder to a GitHub repository
2. Go to [render.com](https://render.com) and sign in
3. Click **New → Static Site**
4. Connect your GitHub repo
5. Settings:
   - **Build Command:** *(leave empty)*
   - **Publish Directory:** `.`
6. Click **Create Static Site** → live in ~1 minute 🚀

## Customize

Before deploying, update these in `index.html`:
- Team member names, roles, bios, and links
- Project links (GitHub + live URLs)
- Contact email, WhatsApp, and socials
- Hero stats (projects delivered, years, etc.)
